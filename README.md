# University Database Management System

## Overview
This project is a SQL Server database system designed to manage university-related data such as students, instructors, courses, departments, classrooms, enrollments, and grades.

The project demonstrates both fundamental and advanced SQL concepts through practical queries and database objects.

## Features
- Student and instructor management
- Course and department records
- Enrollment tracking
- Classroom scheduling
- Grade management

## SQL Concepts Implemented
- Operators (Arithmetic, Comparison, Logical)
- Joins (Inner, Left, Right, Full, Self)
- Subqueries
- CASE Statements
- Date Functions
- Indexes
- Views
- Window Functions
- Common Table Expressions (CTE)
- Stored Procedures
- User Defined Functions
- Triggers

## Technologies
- Microsoft SQL Server
- T-SQL

## Learning Outcomes
This project helped develop practical skills in database design, query optimization, data retrieval, and SQL Server programming.
